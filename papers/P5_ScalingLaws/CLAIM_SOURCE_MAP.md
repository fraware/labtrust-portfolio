# P5 claim source map (canonical paths)

Every numeric claim in the NeurIPS submission should trace to this map and to `papers/P5_ScalingLaws/generated_tables.md` (regenerated by `scripts/export_scaling_tables.py`).

## Canonical artifact paths

| Artifact | Path |
|----------|------|
| Multiscenario runs | `datasets/runs/multiscenario_runs/` |
| Scenario LOO | `datasets/runs/scaling_eval/heldout_results.json` |
| Family LOO | `datasets/runs/scaling_eval_family/heldout_results.json` |
| Regime LOO | `datasets/runs/scaling_eval_regime/heldout_results.json` |
| Agent-count LOO | `datasets/runs/scaling_eval_agent_count/heldout_results.json` |
| Fault-setting LOO | `datasets/runs/scaling_eval_fault/heldout_results.json` |
| Seed sensitivity | `datasets/runs/sensitivity_sweep/scaling_sensitivity.json` |
| Recommendation | `datasets/runs/scaling_recommend/recommendation_eval.json` |
| Regime × agent summary | `datasets/runs/scaling_summary/regime_agent_summary.json` |
| Exported tables | `papers/P5_ScalingLaws/generated_tables.md` |
| Depth-≤2 run-tree hashes | `papers/P5_ScalingLaws/p5_artifact_hashes.txt` (from `scripts/hash_p5_artifacts.py`) |

Produced by `PYTHONPATH=impl/src LABTRUST_KERNEL_DIR=kernel python scripts/run_paper_experiments.py --paper P5` and `scripts/export_scaling_tables.py` (see `papers/P5_ScalingLaws/README.md`).

## Required source map (YAML)

Machine-readable twin: `papers/P5_ScalingLaws/claim_sources.yaml` (used by `scripts/check_p5_claim_sources.py`).

```yaml
claims:
  grid_size:
    value: 7200
    source: datasets/runs/scaling_eval/heldout_results.json
    field: total_rows

  scenario_holdout_mae:
    source: datasets/runs/scaling_eval/heldout_results.json
    paper_location: Main Table 2

  family_holdout_mae:
    source: datasets/runs/scaling_eval_family/heldout_results.json
    paper_location: Main Table 3

  regime_agent_deltas:
    source: datasets/runs/scaling_summary/regime_agent_summary.json
    paper_location: Main Table 1

  seed_sensitivity:
    source: datasets/runs/sensitivity_sweep/scaling_sensitivity.json
    paper_location: Appendix

  recommendation_metrics:
    source: datasets/runs/scaling_recommend/recommendation_eval.json
    paper_location: Appendix only

  collapse_train_prevalence_brier:
    source: datasets/runs/scaling_eval/heldout_results.json
    fields: overall_collapse_rate, overall_collapse_brier_train_prevalence
    paper_location: Appendix (collapse table)
```

## Trigger semantics reminder

- **Admissible** baselines drive `success_criteria_met.trigger_met` (global mean, num-tasks bucket, regime train mean).
- **Oracle** `oracle_baselines.per_scenario_mean_including_test_mae` is diagnostic only.
