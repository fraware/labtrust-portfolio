# P5 — Scaling Laws (Empirical Predictors)

P5 contributes scenario-based coordination tax prediction with held-out validation and comparison to prior model. It studies empirical predictors of coordination tax: task features (scenario, num_tasks, faults), baseline hierarchy (global mean, per-scenario mean, regression), and held-out scenario MAE. Draft and claims: [DRAFT.md](DRAFT.md), [claims.yaml](claims.yaml). Tables and figures: [VISUALS_PER_PAPER.md](../docs/VISUALS_PER_PAPER.md), [RESULTS_PER_PAPER.md](../docs/RESULTS_PER_PAPER.md). Outline: [AUTHORING_PACKET.md](AUTHORING_PACKET.md). DRAFT includes Results summary (excellence metrics), Threats to validity, and per-claim confidence. Conditional paper: trigger in heldout_results.json (beat_per_scenario_baseline, trigger_met); see [CONDITIONAL_TRIGGERS.md](../docs/CONDITIONAL_TRIGGERS.md) and [STANDARDS_OF_EXCELLENCE.md](../docs/STANDARDS_OF_EXCELLENCE.md). To run with other papers: `python scripts/run_paper_experiments.py --paper P5`. Key results (MAE, trigger_met, R²) are in Section 5 of DRAFT and in [RUN_RESULTS_SUMMARY.md](../../datasets/runs/RUN_RESULTS_SUMMARY.md) after running the publishable pipeline.
