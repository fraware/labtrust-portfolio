# P4 — CPS-MAESTRO

P4 defines a benchmark and fault-injection suite for CPS-grade agent coordination: scenario-driven runs, drop/delay faults, and standardized scoring (tasks_completed, p95 latency, variance). **Adapters:** Centralized, Blackboard, RetryHeavy (architecturally distinct; per-task retries). Fault-sweep reports include recovery metrics (steps_to_completion_after_first_fault, tasks_completed_after_fault). Draft and claims: [DRAFT.md](DRAFT.md), [claims.yaml](claims.yaml). Tables and figures: [VISUALS_PER_PAPER.md](../docs/VISUALS_PER_PAPER.md), [RESULTS_PER_PAPER.md](../docs/RESULTS_PER_PAPER.md). Outline: [AUTHORING_PACKET.md](AUTHORING_PACKET.md). MAESTRO adoption: [bench/maestro/README.md](../../bench/maestro/README.md). To run with other papers: `python scripts/run_paper_experiments.py --paper P4`. Results summary: [RUN_RESULTS_SUMMARY.md](../../datasets/runs/RUN_RESULTS_SUMMARY.md).
