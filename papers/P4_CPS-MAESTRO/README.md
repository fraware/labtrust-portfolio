# P4 — CPS-MAESTRO

P4 is a **lab-first** scenario benchmark for agent coordination under explicit faults, with **MAESTRO_REPORT v0.2** (recovery timing, safety counters, run_outcome) and safety-dominant composite scoring. Auxiliary micro-scenarios (`warehouse_v0`, `traffic_v0`) plus regime stress complement lab profiles but are not equal-strength pillars. **Adapters:** Centralized, Blackboard (parameterization), RetryHeavy, NoRecovery, ConservativeSafeShutdown. Draft and claims: [DRAFT.md](DRAFT.md), [claims.yaml](claims.yaml). Machine tables: [generated_tables.md](generated_tables.md). Frozen evidence: `datasets/releases/p4_publishable_v1/` (regenerated and synchronized 2026-04-20). Synced convenience artifacts remain available under `datasets/runs/maestro_fault_sweep/`, `datasets/runs/maestro_antigaming/`, and `bench/maestro/` for reproducibility workflows. MAESTRO adoption: [bench/maestro/README.md](../../bench/maestro/README.md). Summary: [RUN_RESULTS_SUMMARY.md](../../datasets/runs/RUN_RESULTS_SUMMARY.md).
