# P6 — Safe LLM Planning

P6 contributes a CPS-oriented typed-plan firewall with deterministic capture, red-team + confusable deputy + jailbreak-style suite aligned with OWASP, **real-LLM table (Table 1b)** with 5 runs per case (pass_rate_pct, 95% Wilson CI, latency mean ± stdev), **3-way baseline comparison** (gated / weak / ungated), and denial-trace statistics. It defines a typed-plan firewall and validator stack for CPS: plan schema, red-team suite (8 cases), confusable deputy (4), jailbreak-style cases, and MAESTRO adapter with latency reporting (3 scenarios, 20 seeds publishable default). Validator v0.2: allow_list + safe_args (path traversal, dangerous patterns, jailbreak-style phrases). OWASP LLM Top 10 coverage: [docs/P6_OWASP_MAPPING.md](../docs/P6_OWASP_MAPPING.md). Draft and claims: [DRAFT.md](DRAFT.md), [claims.yaml](claims.yaml). Tables and figures: [VISUALS_PER_PAPER.md](../docs/VISUALS_PER_PAPER.md), [RESULTS_PER_PAPER.md](../docs/RESULTS_PER_PAPER.md). Outline: [AUTHORING_PACKET.md](AUTHORING_PACKET.md). Full results and commands: [P6_RESULTS_REPORT.md](P6_RESULTS_REPORT.md). Firewall is planner-agnostic; real LLM via .env and `--real-llm --real-llm-runs 5`. Conditional trigger: success_criteria_met.trigger_met; see [CONDITIONAL_TRIGGERS.md](../docs/CONDITIONAL_TRIGGERS.md). To run: `python scripts/run_paper_experiments.py --paper P6`. Key results: 8/8 synthetic red-team, 4/4 confusable deputy, jailbreak-style pass; real-LLM 91.7% pass (55/60 runs, CI [81.9, 96.4]); adapter latency; 3-way baseline (tool-level and argument-level); denial_trace_stats. Artifacts: red_team_results.json (real_llm when --real-llm), baseline_comparison.json (--run-baseline), baseline_comparison_args.json (--run-baseline --baseline-plan args_unsafe), denial_trace_stats.json (--run-adapter --denial-stats); export: export_llm_redteam_table.py, export_p6_baseline_table.py [--baseline-file baseline_comparison_args.json], export_p6_artifact_hashes.py (Appendix A). **Venue pack (SaT-CPS 2026):** [sat-cps2026/](sat-cps2026/) with submission standard, roles, runbook, ACM main.tex.
