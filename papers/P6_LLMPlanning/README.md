# P6 — Safe LLM Planning

P6 contributes a CPS-oriented typed-plan firewall with deterministic capture, expanded red-team + confusable deputy + jailbreak-style suites aligned with OWASP, **real-LLM table (Table 1b)** with Wilson CI, **3-way baseline comparison** (gated / weak / ungated) plus benign utility suite, and denial-trace statistics. It defines a typed-plan firewall and validator stack for CPS: plan schema, red-team suite (15 cases), confusable deputy (6), jailbreak-style cases (4), adaptive suite, and MAESTRO adapter with latency reporting (3 scenarios, 20 seeds publishable default). Validator stack: allow_list + safe_args + ponr_gate + privilege heuristic. OWASP LLM Top 10 coverage: [docs/P6_OWASP_MAPPING.md](../docs/P6_OWASP_MAPPING.md). Draft and claims: [DRAFT.md](DRAFT.md), [claims.yaml](claims.yaml). Tables and figures: [VISUALS_PER_PAPER.md](../docs/VISUALS_PER_PAPER.md), [RESULTS_PER_PAPER.md](../docs/RESULTS_PER_PAPER.md). Outline: [AUTHORING_PACKET.md](AUTHORING_PACKET.md). Full results and commands: [P6_RESULTS_REPORT.md](P6_RESULTS_REPORT.md). Firewall is planner-agnostic; real LLM via `.env` and `llm_redteam_eval.py --real-llm` (default OpenAI; `--real-llm-provider auto|openai|anthropic|prime`). **Current Table 1b camera-ready snapshot (`datasets/runs/llm_eval_camera_ready_20260424`, OpenAI):** `gpt-4.1-mini`, `gpt-4.1`, full-suite mode (25 cases/model), 3 runs/case: 75/75 each (100.0%, Wilson [95.1, 100.0]). **Canonical command for this snapshot:** `--out datasets/runs/llm_eval_camera_ready_20260424 --real-llm-runs 3 --real-llm-suite full`. **Optional:** Prime Inference multi-model matrix in a separate output directory; see [sat-cps2026/EXPERIMENTS_RUNBOOK.md](sat-cps2026/EXPERIMENTS_RUNBOOK.md). See [EVALS_RUNBOOK.md](../../docs/EVALS_RUNBOOK.md) P6. Conditional trigger: success_criteria_met.trigger_met; see [CONDITIONAL_TRIGGERS.md](../docs/CONDITIONAL_TRIGGERS.md). To run: `python scripts/run_paper_experiments.py --paper P6` or `python scripts/p6_publish_bundle.py`. Key results: 15/15 synthetic red-team, 6/6 confusable deputy, 4/4 jailbreak-style; adapter latency p95 mean ~32 ms (3 scenarios x 20 seeds); 3-way baseline (tool-level denials 60/60/0 gated/weak/ungated; argument-level gated-only denials for path traversal; benign suite optional).
