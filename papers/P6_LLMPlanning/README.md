# P6 — Safe LLM Planning

P6 contributes a CPS-oriented typed-plan firewall with deterministic capture, red-team + confusable deputy + jailbreak-style suite aligned with OWASP, **real-LLM table (Table 1b)** with Wilson CI, **3-way baseline comparison** (gated / weak / ungated) plus benign utility suite, and denial-trace statistics. It defines a typed-plan firewall and validator stack for CPS: plan schema, red-team suite (9 cases), confusable deputy (4), jailbreak-style cases, and MAESTRO adapter with latency reporting (3 scenarios, 20 seeds publishable default). Validator v0.2: allow_list + safe_args (path traversal, dangerous patterns, jailbreak-style phrases). OWASP LLM Top 10 coverage: [docs/P6_OWASP_MAPPING.md](../docs/P6_OWASP_MAPPING.md). Draft and claims: [DRAFT.md](DRAFT.md), [claims.yaml](claims.yaml). Tables and figures: [VISUALS_PER_PAPER.md](../docs/VISUALS_PER_PAPER.md), [RESULTS_PER_PAPER.md](../docs/RESULTS_PER_PAPER.md). Outline: [AUTHORING_PACKET.md](AUTHORING_PACKET.md). Full results and commands: [P6_RESULTS_REPORT.md](P6_RESULTS_REPORT.md). Firewall is planner-agnostic; real LLM via `.env` and `llm_redteam_eval.py --real-llm` (default OpenAI; `--real-llm-provider auto|openai|anthropic|prime`). **Canonical Table 1b (OpenAI):** `gpt-4.1-mini`, `gpt-4.1`, 5 runs/case (2026-03-17 snapshot: 55/65 each, 84.6%, Wilson [73.9, 91.4]; 0/5 on two argument-level cases). **Optional:** Prime Inference four-model matrix (N=3, separate output directory; see [sat-cps2026/EXPERIMENTS_RUNBOOK.md](sat-cps2026/EXPERIMENTS_RUNBOOK.md)). See [EVALS_RUNBOOK.md](../../docs/EVALS_RUNBOOK.md) P6. Conditional trigger: success_criteria_met.trigger_met; see [CONDITIONAL_TRIGGERS.md](../docs/CONDITIONAL_TRIGGERS.md). To run: `python scripts/run_paper_experiments.py --paper P6`. Key results: 9/9 synthetic red-team, 4/4 confusable deputy, jailbreak-style pass; adapter latency p95 mean ~32 ms (3 scenarios x 20 seeds); 3-way baseline (tool-level denials 60/60/0 gated/weak/ungated; argument-level gated-only denials for path traversal; benign suite optional). Artifacts: red_team_results.json (real_llm / real_llm_models when --real-llm), baseline_comparison.json, baseline_comparison_args.json, baseline_benign.json, denial_trace_stats.json, p6_artifact_hashes.json; export: export_llm_redteam_table.py, export_p6_baseline_table.py, export_p6_artifact_hashes.py, export_p6_reproducibility_table.py, export_p6_layer_attribution.py, export_p6_failure_analysis.py, export_p6_cross_model_heatmap.py, export_p6_latency_decomposition.py. **Venue pack (SaT-CPS 2026):** [sat-cps2026/](sat-cps2026/) with submission standard, roles, runbook, ACM main.tex.
