# P8 — Meta-Coordination

P8 studies meta-controller regime switching under stress: threshold-based switch criterion, regime_switch events, and comparison to fixed and naive baselines. **Options:** Optional `--fallback-adapter blackboard|centralized|retry_heavy` for two-regime comparison; `--stress-preset very_high` or scenario `regime_stress_v1.yaml` for second stress scenario. Draft and claims: [DRAFT.md](DRAFT.md), [claims.yaml](claims.yaml). Tables and figures: [VISUALS_PER_PAPER.md](../docs/VISUALS_PER_PAPER.md), [RESULTS_PER_PAPER.md](../docs/RESULTS_PER_PAPER.md). Outline: [AUTHORING_PACKET.md](AUTHORING_PACKET.md). Publishable tables use 20 seeds; for a non-vacuous trigger run meta_collapse_sweep then meta_eval with a drop_prob where collapse_count > 0. DRAFT includes Results summary (excellence metrics), Threats to validity, and per-claim confidence. See [CONDITIONAL_TRIGGERS.md](../docs/CONDITIONAL_TRIGGERS.md) and [STANDARDS_OF_EXCELLENCE.md](../docs/STANDARDS_OF_EXCELLENCE.md). To run with other papers: `python scripts/run_paper_experiments.py --paper P8`. Results summary: [RUN_RESULTS_SUMMARY.md](../../datasets/runs/RUN_RESULTS_SUMMARY.md).
