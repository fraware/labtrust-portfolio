# P8 — Meta-Coordination

P8 contributes auditability (every regime_switch in trace with reason) and PONR invariance; under pre-specified stress it reports **non-inferior paired collapse counts** vs fixed (`meta_non_worse_collapse`, legacy `meta_reduces_collapse`) and **strict** reduction separately (`meta_strictly_reduces_collapse`, McNemar, Wilson CIs). With `--fallback-adapter retry_heavy`, two real coordination algorithms are compared. **Options:** `--scenario regime_stress_v0|regime_stress_v1`, `--fallback-adapter blackboard|centralized|retry_heavy`, `--stress-preset high|very_high`, `--non-vacuous` with documented `stress_selection_policy`.
