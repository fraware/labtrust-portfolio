# P8 — Meta-Coordination

P8 studies meta-controller regime switching under stress: threshold-based switch criterion, regime_switch events, and comparison to fixed and naive baselines. Draft and claims: [DRAFT.md](DRAFT.md), [claims.yaml](claims.yaml). Tables and figures: [VISUALS_PER_PAPER.md](../docs/VISUALS_PER_PAPER.md), [RESULTS_PER_PAPER.md](../docs/RESULTS_PER_PAPER.md).
