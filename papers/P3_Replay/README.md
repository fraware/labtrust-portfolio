# P3 — Replay Levels and Nondeterminism Detection

P3 defines replay levels (L0/L1/L2) and nondeterminism detection for agentic CPS; L0 control-plane replay with state_hash and diagnostics is implemented; L1 stub and L1 twin (--l1-twin); L2 design in kernel. Draft and claims: [DRAFT.md](DRAFT.md), [claims.yaml](claims.yaml). Tables and figures: [VISUALS_PER_PAPER.md](../docs/VISUALS_PER_PAPER.md), [RESULTS_PER_PAPER.md](../docs/RESULTS_PER_PAPER.md). Outline: [AUTHORING_PACKET.md](AUTHORING_PACKET.md). To run with other papers: `python scripts/run_paper_experiments.py --paper P3`. Key results (fidelity, divergence localization, L1 stub, overhead) are in Section 5 of DRAFT and in [RUN_RESULTS_SUMMARY.md](../../datasets/runs/RUN_RESULTS_SUMMARY.md).
