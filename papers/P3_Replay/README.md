# P3 — Replay Levels and Nondeterminism Detection

P3 defines replay levels (L0/L1/L2) and nondeterminism detection for agentic CPS. The reference eval is `scripts/replay_eval.py`, which writes `datasets/runs/replay_eval/summary.json` with `schema_version: p3_replay_eval_v0.2` (full L0 overhead, baselines, multi-seed thin-slice family, corpus outcomes with **localization and space fields**, **`corpus_category`** per trace, **`corpus_space_summary`**, optional **`l1_twin_summary`** when `--l1-twin` is used, optional overhead curve and bootstrap CIs).

**Read first:** [DRAFT.md](DRAFT.md) (repro block, evaluation, limitations), [claims.yaml](claims.yaml), [generated_tables.md](generated_tables.md).

**Scripts:** `export_replay_corpus_table.py --out-md papers/P3_Replay/generated_tables.md` (Table 1 + Table 1b); tables 2–3 from summary JSON fields (also documented in `generated_tables.md`); `generate_p3_replay_corpus_traces.py` (regenerate auxiliary corpus JSON); **`export_p3_paper_figures.py`** (camera-ready PNGs under `papers/P3_Replay/figures/`: levels diagram, overhead curve, baseline bar chart, corpus-lane counts, first-divergence timeline; refreshes `docs/figures/p3_replay_overhead.*` for verification); or run individual generators: `export_p3_replay_levels_diagram.py` (Mermaid `.mmd`), `export_p3_replay_levels_figure.py` (PNG Figure 0), `plot_replay_overhead.py`, `plot_p3_baseline_bars.py`, `plot_p3_corpus_lanes.py`, `export_p3_first_divergence_timeline.py`; `verify_p3_replay_summary.py` (schema, corpus space fields, optional figure consistency).
