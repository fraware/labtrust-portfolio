# P2 — REP-CPS

P2 defines a safety-gated CPS profile for authenticated sensitivity-sharing (typed variables, freshness, rate limits, provenance, robust aggregation) integrated with MADS safety gates. Draft and claims: [DRAFT.md](DRAFT.md), [claims.yaml](claims.yaml). Tables and figures: [VISUALS_PER_PAPER.md](../docs/VISUALS_PER_PAPER.md), [RESULTS_PER_PAPER.md](../docs/RESULTS_PER_PAPER.md). Outline: [AUTHORING_PACKET.md](AUTHORING_PACKET.md). **Eval:** Default publishable uses two scenarios (toy_lab_v0, lab_profile_v0); optional `--aggregation-steps 5` for multi-step aggregation; profile ablation in summary.json `profile_ablation` (Table 5 in generated_tables.md). Conditional paper: in the evaluated scenario, sensitivity sharing does not materially change tasks_completed; the contribution is the profile and MAESTRO-compatible harness. See [CONDITIONAL_TRIGGERS.md](../docs/CONDITIONAL_TRIGGERS.md) and [STANDARDS_OF_EXCELLENCE.md](../docs/STANDARDS_OF_EXCELLENCE.md). To run: `python scripts/run_paper_experiments.py --paper P2`. Key results (adapter parity, reduced observed bias under compromise, safety gate) are in Section 5 of DRAFT and in [RUN_RESULTS_SUMMARY.md](../../datasets/runs/RUN_RESULTS_SUMMARY.md).
