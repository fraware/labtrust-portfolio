# P2 — REP-CPS

P2 defines a CPS profile for sensitivity-sharing (typed variables, rate limits, robust aggregation) integrated with MADS safety gates. Draft and claims: [DRAFT.md](DRAFT.md), [claims.yaml](claims.yaml). Tables and figures: [VISUALS_PER_PAPER.md](../docs/VISUALS_PER_PAPER.md), [RESULTS_PER_PAPER.md](../docs/RESULTS_PER_PAPER.md). Outline: [AUTHORING_PACKET.md](AUTHORING_PACKET.md). **Eval:** Default publishable uses two scenarios (toy_lab_v0, lab_profile_v0); optional `--aggregation-steps 5` for multi-step aggregation; convergence metrics and Table 4 via `python scripts/export_rep_cps_convergence_table.py`. DRAFT includes Results summary (excellence metrics), Threats to validity, and per-claim confidence. Conditional paper: trigger requires REP-CPS to improve robustness on at least one non-toy scenario; summary reports `success_criteria_met.trigger_met`. See [CONDITIONAL_TRIGGERS.md](../docs/CONDITIONAL_TRIGGERS.md) and [STANDARDS_OF_EXCELLENCE.md](../docs/STANDARDS_OF_EXCELLENCE.md). To run with other papers: `python scripts/run_paper_experiments.py --paper P2`. Key results (adapter parity, bias reduction, trigger_met) are in Section 5 of DRAFT and in [RUN_RESULTS_SUMMARY.md](../../datasets/runs/RUN_RESULTS_SUMMARY.md).
