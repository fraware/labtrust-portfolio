# Visuals Per Paper (Peer-Review Standard)

This document defines the **full set of tables and figures** expected for each paper to meet peer-review publication standards in systems, assurance, and coordination venues. A publishable paper typically includes: (1) a **system or methodology overview figure**, (2) **at least two result tables**, (3) **at least one result figure** (data-generated), and (4) where applicable a **comparison or taxonomy table**. All visuals must be **autogenerated from repo artifacts** via documented scripts.

## Output paths

Overview diagrams (Figure 0) are written to `docs/figures/` as `.mmd` (Mermaid) files so they can be versioned and rendered for camera-ready drafts. For P1 camera-ready, `render_p1_flow_figure.py` stages the final PDF/PNG in `papers/P1_Contracts/figures/` and mirrors PNG to `docs/figures/`. Result figures remain `.png` (or `.json` sidecars when matplotlib is unavailable). Tables are regenerated via export scripts and pasted into the draft or consumed by a build step. **Captions and how to read:** Table and figure captions and "how to read" text live in each paper's DRAFT.md and, for generated tables, in `papers/Px_.../generated_tables.md`; see [PAPER_ARTIFACT_STYLE.md](PAPER_ARTIFACT_STYLE.md) for caption format.

## Reproducibility

Each paper's repro block at the top of DRAFT.md must list the exact command for **every** table and figure, including Figure 0. Example:

- Figure 1: `python scripts/export_p0_assurance_pipeline.py` (output `docs/figures/p0_assurance_pipeline.mmd`).
- Table 1: `python scripts/build_p0_conformance_corpus.py`, then `python scripts/export_e1_corpus_table.py`.
- Table 2: `python scripts/e2_redaction_demo.py`, then `python scripts/export_e2_admissibility_matrix.py`.
- Table 3: `python scripts/run_p0_e4_controller_matrix.py --seeds 20 --scenarios toy_lab_v0,lab_profile_v0 --regimes baseline,moderate,stress --out datasets/runs/p0_e4_controller_matrix.json`, then `python scripts/export_p0_table3.py` (strong replay preference: E4 raw baseline rows, then E3 strong replay).
- Figure 2: `python scripts/export_p0_tier_lattice.py`. Figure 3: `python scripts/export_p0_redaction_figure.py`.
- Figure (E3 latency): `python scripts/plot_e3_latency.py`.
